# Images Directory

Please upload your images here with the following names:

- `hero.jpg` - Hero section image (team photo)
- `team.jpg` - Team photo for carousel
- `workshop.jpg` - Workshop image for carousel/gallery
- `community.jpg` - Community event image
- `outreach.jpg` - Rural outreach image
- `school.jpg` - School visit image

The application will automatically use these images in:
- Home page hero section
- About page carousel
- Gallery page masonry grid

All images should be optimized for web (recommended: max 1920px width, JPEG format).
